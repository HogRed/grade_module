# Grade Module (IS-2100 Demo)

This module demonstrates basic interaction with Git / GitHub / GitHub Desktop and how to build a small program in Python that uses the script and module style of building applications.

This repository consists of two files, grade_utils and main. grade_utils has functions for analyzing dictionaries of student grade information.

## How to Run

To run this file, navigate to it and run:

```bash
python main.py
```
